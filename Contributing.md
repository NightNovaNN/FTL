# Thanks For Contributing to FTL

## FTL Contributing

FTL has 2 main methods of contributing:
- Subsystem Contributions
- General Contributions

### Subsystem Contributions

Here you can contribute to make the Assembly portable to any device, if you're interested, check `src/asm/READ.md`.

### General Contributions

This is the contribution type everywhere, here you can make the code more readable, or better quality.  
If you're a developer with experience or knowledge in `Fortran` or `C` or `Assembly`, you should contribute to general.

---

## Methods

Steps to Contribute:
- Fork the repo on `Github`.
- Add your changes or improvements.
- Add a `Pull Request` to request your changes to be included.
- If successful, the code will be merged.
